# advent-of-code-2025

To download your puzzle inputs, get your session cookie from the browser when logged in at [Advent of Code](https://adventofcode.com). Save it in an environment variable named `AOC_COOKIE`.

```bash
curl --cookie session=$AOC_COOKIE [input url] > input
```

For example, for day 1 do:

```bash
curl --cookie session=$AOC_COOKIE https://adventofcode.com/2025/day/1/input > input
```

To run the solutions do:

```
./run.awk input
```
